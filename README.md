# Interactive Sales Dashboard

## 📌 Objective
Build an interactive dashboard to visualize KPIs like sales, profit, orders, and category performance.

## 📊 Dataset
- Source: [Superstore Dataset](https://www.kaggle.com/datasets/juhi1994/superstore)

## 🛠️ Tools & Libraries
- Power BI, Plotly, Tableau, Pandas

## 🚀 Steps
1. Load and preprocess dataset.
2. Compute KPIs (sales, profit, orders).
3. Build visualizations (bar, line, pie, map).
4. Add filters (region, category, date).
5. Publish dashboard.

## 📈 Deliverables
- KPI dashboard
- Filter-enabled charts
- GitHub documentation
